# Mi Portafolio Web

Este es un proyecto construido desde cero durante un ciclo de desarrollo web frontend, enfocado en dominar las bases de maquetación semántica, diseño responsivo avanzado y manipulación interactiva del DOM.

## 🚀 Demos en Vivo
Puedes ver el proyecto desplegado y completamente funcional en cualquiera de los siguientes enlaces:
* [Despliegue en Netlify](https://ozkrtr-mi-portafolio.netlify.app/)
* [Despliegue en GitHub Pages](https://ozkrtr.github.io/)

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Arquitectura y estructura semántica.
* **CSS3 Avanzado:** Flexbox, CSS Grid autoadaptable (`minmax`, `auto-fit`), variables globales (`Design Tokens`) y microinteracciones (`transitions`).
* **JavaScript (Vanilla):** Captura de eventos del DOM, validación defensiva de formularios y persistencia de datos local (`LocalStorage`).

## ⚙️ Características Principales
* **Diseño Mobile-First:** Completamente adaptado a dispositivos móviles y escritorios sin romper la simetría.
* **Componente de Mensajería:** Formulario interactivo que valida entradas en tiempo real y renderiza dinámicamente un feed de mensajes guardados en el navegador.
